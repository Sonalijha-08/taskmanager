# Task Manager Pro 🚀

A full-stack, automated task management application designed for productivity with a touch of celebration.

## ✨ Features

- **Full CRUD Functionality**: Create, Read, Update, and Delete tasks with ease.
- **Dynamic Task Management**:
  - Filter and organize tasks by category.
  - Set due dates to keep track of deadlines.
  - Mark tasks as complete with a satisfying UI feedback.
- **Celebration Effects**: Completing a task triggers a burst of confetti and motivational praises to boost your productivity.
- **Real-time Validation**: Built-in validation ensures task titles are present and character limits are respected.
- **Responsive Design**: Modern, dark-themed UI that looks great on all devices.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vanilla CSS, Canvas-Confetti
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- MongoDB instance (local or Atlas)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sonalijha-08/taskmanager.git
   cd task-manager
   ```

2. Install dependencies for both client and server:
   ```bash
   npm run install-all
   ```

3. Configure environment variables:
   - Create a `.env` file in the `server` directory.
   - Add `MONGODB_URI` and `PORT`.

4. Start the application:
   ```bash
   npm run dev
   ```
